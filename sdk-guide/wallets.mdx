## Prerequisites

Every crypto operation requires:

1. `shareStore` configured on `TrustodianSDK`
2. `const userRef = await sdk.getSdkUserId()`
3. `await sdk.unlockWalletSession(userRef, password)` (or `setWalletPassword` on first use)
4. `userRef` on create/import/send/export (or `runWithUserRef`)

See [Authentication](/sdk-guide/authentication) and [Key Management](/sdk-guide/key-management).

## Creating Wallets

```typescript
import { TrustodianSDK, InMemoryShareStore, EChainType } from 'vaults-multichain-sdk';

await sdk.unlockWalletSession(userRef, password);

// Local key generation (default deployment)
const { wallets } = await sdk.createWallet({
  chain: EChainType.EVM,
  userRef,
  password,
  mode: 'generate',
});

// Server mnemonic + ECDH (when getAppConfig().genWalletFromMnemonic === true)
const { wallets: ecdhWallets } = await sdk.createWallet({
  chain: EChainType.EVM,
  userRef,
  password,
  mode: 'ecdh',
});
```

## Importing Wallets

Private key is split **locally** — never uploaded to the server.

```typescript
await sdk.importWallet({
  chain: EChainType.EVM,
  privateKey: '0x...',
  userRef,
  password,
});
```

## Reading Wallets

```typescript
// Paginated list
const { wallets, total, page, limit } = await sdk.getWallets({
  page: 1,
  limit: 20,
  chainTypes: [EChainType.EVM, EChainType.SOL],
});

// Single wallet by chain + address
const wallet = await sdk.getWallet({
  chain: EChainType.EVM,
  address: '0x...',
});
```

## Sending Transactions

Requires unlocked session. Pass `userRef` or use `runWithUserRef`.

```typescript
await sdk.runWithUserRef(userRef, async () => {
  const { txHash } = await sdk.sendTransaction({
    chain: EChainType.EVM,
    from: senderAddress,
    to: recipientAddress,
    amount: '0.1',
    chainId: 1,
    tokenAddress: '',
    decimals: 18,
    isNativeToken: true,
  });
});
```

For **organization-owned** wallets, add `walletScope: 'organization'` — the SDK routes share fetches to `/sdk/organization/wallets/...`.

### New device / missing device share

Pass `walletPassword` to trigger recovery rotation:

```typescript
await sdk.sendTransaction({
  userRef,
  walletPassword: password,
  chain: EChainType.EVM,
  from,
  to,
  amount: '0.1',
  chainId: 1,
  tokenAddress: '',
  decimals: 18,
  isNativeToken: true,
});
```

## Portfolio & Balances

```typescript
// Single wallet
const portfolio = await sdk.getPortfolio({
  chain: EChainType.EVM,
  address: walletAddress,
  chainId: 1,
});
// portfolio.tokens[].balanceUSD, priceUSD

// Batch
const portfolios = await sdk.getWalletsPortfolio([
  { walletAddress: '0x...', chainType: EChainType.EVM, chain: '1' },
  { walletAddress: '...', chainType: EChainType.SOL, chain: 'sol_mainnet' },
]);

const { totalAssetValueUSD } = await sdk.getTotalAssetValue();
```

```typescript
const history = await sdk.getWalletHistory({

```

## Fee Estimation

```typescript
import { TransactionEstimatingType } from 'vaults-multichain-sdk';

const fee = await sdk.estimateFee({
  chain: EChainType.EVM,
  txType: TransactionEstimatingType.Send,
  payload: { from, to, amount: '0.1' },
  chainId: 1,
});
```

## Export Private Key

```typescript
const { privateKey } = await sdk.exportWallet({
  userRef,
  chain: EChainType.EVM,
  address: walletAddress,
  format: 'hex',
});
```

<Warning>
  Handle exported keys securely. Prefer never exporting in production integrations.
</Warning>

## Rename & Delete

```typescript
await sdk.updateWalletName({
  chain: EChainType.EVM,
  address: walletAddress,
  name: 'Treasury',
});

await sdk.deleteWallet({
  chain: EChainType.EVM,
  address: walletAddress,
});
```
