Cowchain provides the `vaults-multichain-sdk` TypeScript library so your backend can create wallets, manage multisig vaults, and send transactions across six blockchains — all without your server ever holding a complete private key. Every wallet uses 2-of-3 Shamir secret sharing: you control the device share, Trustodian holds one encrypted share, and a recovery share is stored separately. No single party can spend funds alone.

## How it works

<Steps>
  <Step title="Install the SDK">
    Add `vaults-multichain-sdk` to your Node.js backend.

    ```bash
    npm install vaults-multichain-sdk
    ```
  </Step>
  <Step title="Get an API key">
    Sign in to the [Trustodian Dashboard](https://app.trustodian.com), go to **Settings → API Keys**, and create a key with the permissions your integration needs.
  </Step>
  <Step title="Initialize the SDK">
    Create a `TrustodianSDK` instance with your API key and a share store for persisting device shares.

    ```typescript
    import { TrustodianSDK, InMemoryShareStore, EChainType } from 'vaults-multichain-sdk';

    const sdk = new TrustodianSDK({
      apiKey: process.env.TRUSTODIAN_API_KEY!,
      network: 'mainnet',
      shareStore: new InMemoryShareStore(), // replace with DB adapter in production
    });
    ```
  </Step>
  <Step title="Create your first wallet">
    Set a wallet password and generate a non-custodial wallet. The private key is split locally and never sent to the server.

    ```typescript
    const userRef = await sdk.getSdkUserId();
    await sdk.setWalletPassword({ userRef, password: walletPassword });

    const { wallets } = await sdk.createWallet({
      chain: EChainType.EVM,
      userRef,
      password: walletPassword,
      mode: 'generate',
    });

    console.log('Wallet address:', wallets[0].address);
    ```
  </Step>
</Steps>

## Supported chains

| Chain | Key | Networks |
|-------|-----|----------|
| **EVM** | `EChainType.EVM` | Ethereum, Base, Arbitrum, Optimism, Polygon, BNB, Avalanche + testnets |
| **Solana** | `EChainType.SOL` | Mainnet, devnet |
| **Bitcoin** | `EChainType.BTC` | Mainnet, testnet |
| **Tron** | `EChainType.TRON` | Mainnet, Shasta |
| **Cardano** | `EChainType.ADA` | Mainnet, preprod |
| **XRP** | `EChainType.XRP` | Mainnet, testnet |
