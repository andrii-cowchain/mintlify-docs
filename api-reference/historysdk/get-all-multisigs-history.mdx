---
openapi: get /sdk/history/multisigs
title: "Get Transaction History Across All Multisig Wallets"
sidebarTitle: "All Multisigs History"
description: "GET /sdk/history/multisigs - Returns aggregated transaction history across all multisig wallets owned by the authenticated user, ordered by timestamp."
---

Retrieve the combined transaction history for every multisig wallet owned by the authenticated user. The response aggregates records from all associated multisig wallets and returns them in reverse chronological order. Each record includes the multisig ID, transaction hash, signers, status, and timestamp.
