---
openapi: get /sdk/history/{id}
title: "Retrieve the Full Transaction History for a Multisig"
sidebarTitle: "Multisig History"
description: "GET /sdk/history/{id} - Returns the full transaction history for a specific multisig wallet by its ID, including signers and confirmation status."
---

Retrieve the full transaction history for a single multisig wallet using its unique identifier. Supply the multisig `id` as a path parameter. The response lists all transactions associated with that multisig, including each transaction's hash, required signers, collected signatures, confirmation status, and timestamp.
