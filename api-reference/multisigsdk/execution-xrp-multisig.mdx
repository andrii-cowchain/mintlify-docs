openapi: patch /sdk/multisig/execute/xrp/{id}
---

Submit and execute a signed XRP Ledger multisig transaction. Provide the multisig record ID as a path parameter and include the collected signer signatures and transaction payload in the request body to broadcast the transaction to the XRP network.
