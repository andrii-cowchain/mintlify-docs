description: "POST /sdk/multisig/update/name — Update the display name of a multisig wallet by providing the multisig record ID and the new name in the request body."
openapi: post /sdk/multisig/update/name
---

Update the display name of an existing multisig wallet. Submit the multisig identifier and the desired new name in the request body. This operation only modifies the human-readable label and does not alter the on-chain wallet configuration.
