---
title: "Get All User Multisig Wallets Filtered by Chain Type"
sidebarTitle: "Get by Chain"
description: "GET /sdk/multisig/{chainType} — Retrieve all multisig wallets belonging to the authenticated user, filtered by a specific blockchain chain type."
openapi: get /sdk/multisig/{chainType}
---

Retrieve all multisig wallets that belong to the authenticated user, filtered by a specific chain type. Provide the desired chain type as a path parameter to receive all associated multisig records for that network.
