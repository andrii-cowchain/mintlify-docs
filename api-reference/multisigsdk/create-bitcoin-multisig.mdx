---
title: "Create a New Bitcoin Multisig Wallet for a Given Chain"
sidebarTitle: "Create Bitcoin Multisig"
description: "POST /sdk/multisig/create/btc/{chain}/{creatorAddress} — Create a Bitcoin multisig wallet for a given chain and creator address with signer configuration."
openapi: post /sdk/multisig/create/btc/{chain}/{creatorAddress}
---

Create a new Bitcoin multisig wallet by specifying the target chain and the creator's wallet address as path parameters. Provide the signer public keys and threshold in the request body to register and configure the Bitcoin multisig.
