---
title: "Deploy a Previously Created Tron Multisig Wallet On-Chain"
sidebarTitle: "Deploy Tron Multisig"
description: "PATCH /sdk/multisig/execute/tron/{id} — Deploy a created Tron multisig wallet on-chain by providing the record ID and required execution transaction data."
openapi: patch /sdk/multisig/execute/tron/{id}
---

Deploy a previously created Tron multisig wallet on-chain. Provide the multisig record ID as a path parameter and include the required execution transaction data in the request body to broadcast the deployment to the Tron network.
