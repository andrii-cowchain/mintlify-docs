---
title: "Share a Tron Multisig Signing Key with an Invited Co-Signer"
sidebarTitle: "Share Tron Key"
description: "POST /sdk/multisig/share/tron — Distribute Tron multisig key material to an invited signer by submitting the multisig ID and recipient details."
openapi: post /sdk/multisig/share/tron
---

Share Tron multisig key material with an invited co-signer. Submit the multisig identifier and the recipient's details in the request body. The endpoint distributes the necessary key data so the new participant can co-sign Tron multisig transactions.
