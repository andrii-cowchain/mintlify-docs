---
title: "Share a Solana Multisig Signing Key with a Co-Signer"
sidebarTitle: "Share Solana Key"
description: "POST /sdk/multisig/share/sol — Distribute Solana multisig key material to an invited signer by submitting the multisig ID and recipient details."
openapi: post /sdk/multisig/share/sol
---

Share Solana multisig key material with an invited co-signer. Submit the multisig identifier and the recipient's details in the request body. The endpoint distributes the necessary key data so the new participant can co-sign Solana multisig transactions.
