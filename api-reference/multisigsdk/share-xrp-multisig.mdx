title: "Share an XRP Ledger Multisig Configuration with a Signer"
sidebarTitle: "Share XRP Multisig"
description: "POST /sdk/multisig/share/xrp — Share XRP multisig configuration and key data with an invited co-signer by providing the multisig ID and recipient details."
openapi: post /sdk/multisig/share/xrp
---

Share XRP Ledger multisig configuration and key material with an invited co-signer. Submit the multisig identifier and the recipient's information in the request body so the new participant receives the data required to co-sign XRP multisig transactions.
