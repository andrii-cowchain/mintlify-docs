---
title: "Create a New Cardano Multisig Wallet for a Given Chain"
sidebarTitle: "Create Cardano Multisig"
description: "POST /sdk/multisig/create/ada/{chain}/{creatorAddress} — Create a Cardano multisig wallet for a given chain and creator address with signer configuration."
openapi: post /sdk/multisig/create/ada/{chain}/{creatorAddress}
---

Create a new Cardano multisig wallet by specifying the target chain and the creator's wallet address as path parameters. Include signer keys and threshold settings in the request body to register and configure the Cardano multisig.
