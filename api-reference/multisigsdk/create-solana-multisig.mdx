---
title: "Create a New Solana Multisig Wallet for a Given Chain"
sidebarTitle: "Create Solana Multisig"
description: "POST /sdk/multisig/create/sol/{chain}/{creatorAddress} — Create a Solana multisig wallet for a given chain and creator address with signer configuration."
openapi: post /sdk/multisig/create/sol/{chain}/{creatorAddress}
---

Create a new Solana multisig wallet by specifying the target chain and the creator's wallet address as path parameters. Include the signer addresses and threshold configuration in the request body to complete the multisig setup.
