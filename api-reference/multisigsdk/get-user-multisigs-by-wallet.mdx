---
title: "Get User Multisig Wallets by Wallet Address and Chain"
sidebarTitle: "Get by Wallet"
description: "GET /sdk/multisig/{chainType}/{address} — Retrieve all multisig wallets linked to a specific wallet address for the given chain type."
openapi: get /sdk/multisig/{chainType}/{address}
---

Retrieve all multisig wallets that are associated with a specific user wallet address on a given blockchain. Pass the chain type and the wallet address as path parameters to filter and return the relevant multisig records.
