## Base URL

All SDK routes use the `/sdk` path prefix. Set the host to match your deployment (or pass `apiBaseUrl` in `TrustodianSDK`).

| `network` (SDK config) | Default host |
| ---------------------- | ------------ |
| `mainnet` | `https://stage-api.trustodian.online` |
| `testnet` | `https://afridax-backend-production.up.railway.app` |

Example full URL: `https://stage-api.trustodian.online/sdk/wallets`

## Authentication

SDK endpoints are authenticated with an API key in the `x-api-key` header.

| Header      | Description                                                                       |
| ----------- | --------------------------------------------------------------------------------- |
| `x-api-key` | Your API key obtained from the [Trustodian Dashboard](https://app.trustodian.com) |

## Non-custodial endpoints

Most `/sdk/*` routes are **read-only** (wallets list, portfolio, orders) and need only `x-api-key`.

Routes that return **encrypted key shares** (`/sdk/wallets/share/c/...`, `/sdk/wallets/share/s/...`) or accept **wallet-password proofs** (`/sdk/users/enc-token`, `/sdk/oprf/evaluate`) are part of the non-custodial signing flow. **Use [TrustodianSDK](https://www.npmjs.com/package/vaults-multichain-sdk)** on your backend — it handles share decryption and signing. See the [SDK Guide](/sdk-guide/key-management).

Organization routes live under `/sdk/organization/...` (team, wallets, vaults, orders). Start with [Invite member](/api-reference/organizationsdk/post-organization-team-invite) or the [SDK Guide — Organization](/sdk-guide/organization).

## Examples

<CodeGroup>

```javascript JavaScript
fetch('https://stage-api.trustodian.online/sdk/wallets', {
  method: 'GET',
  headers: {
    'x-api-key': 'your-api-key',
    'Content-Type': 'application/json',
  },
})
  .then(response => response.json())
  .then(data => console.log(data));
```

```bash cURL
curl -X GET "https://stage-api.trustodian.online/sdk/wallets" \
  -H "x-api-key: your-api-key" \
  -H "Content-Type: application/json"
```

```python Python
import requests

headers = {
    "x-api-key": "your-api-key",
    "Content-Type": "application/json"
}

response = requests.get(
    "https://stage-api.trustodian.online/sdk/wallets",
    headers=headers,
)
data = response.json()
```

</CodeGroup>
