---
title: "Accept a Wallet Provisioned to You by an Organization Admin"
sidebarTitle: "Accept Wallet"
description: "POST /sdk/organization/wallets/{walletId}/accept — Accept a wallet provisioned by an org admin, completing the assignment flow and granting full access."
openapi: post /sdk/organization/wallets/{walletId}/accept
---

Accept a wallet that has been provisioned to the authenticated team member by an organization admin. Provide the wallet ID as a path parameter. Calling this endpoint completes the wallet assignment flow and grants the user full access to the provisioned wallet.
