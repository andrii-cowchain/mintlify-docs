---
title: "Assign One or More Wallets to an Organization Team Member"
sidebarTitle: "Assign Wallets"
description: "POST /sdk/organization/team/{memberId}/wallets — Assign one or more wallets to a team member by providing the member ID and a list of wallet IDs."
openapi: post /sdk/organization/team/{memberId}/wallets
---

Assign one or more wallets to a specific organization team member. Provide the member's ID as a path parameter and submit a list of wallet IDs in the request body. The assigned wallets become accessible to the specified team member within the organization.
