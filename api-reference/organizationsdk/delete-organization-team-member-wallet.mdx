---
title: "Remove a Wallet Assignment from an Organization Team Member"
sidebarTitle: "Remove Member Wallet"
description: "DELETE /sdk/organization/team/{memberId}/wallets/{walletId} — Revoke a wallet assignment from a team member by member ID and wallet ID without deleting it."
openapi: delete /sdk/organization/team/{memberId}/wallets/{walletId}
---

Remove a specific wallet assignment from an organization team member. Provide the member ID and the wallet ID as path parameters. This revokes the member's access to the specified wallet without deleting the wallet itself.
