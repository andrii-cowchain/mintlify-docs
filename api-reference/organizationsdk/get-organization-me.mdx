---
title: "Get the Organization Profile for the Authenticated User"
sidebarTitle: "Get My Org"
description: "GET /sdk/organization/me — Retrieve the organization profile tied to the current API key, including organization ID, name, and configuration details."
openapi: get /sdk/organization/me
---

Retrieve the organization profile associated with the currently authenticated user. The response includes the organization ID, name, plan details, and relevant configuration metadata for the account making the request.
