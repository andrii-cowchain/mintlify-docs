---
title: "Provision and Activate a Wallet for an Organization Member"
sidebarTitle: "Provision Wallet"
description: "POST /sdk/organization/team/{memberId}/wallets/{walletId}/provision — Provision a wallet for a team member by member ID and wallet ID, activating access."
openapi: post /sdk/organization/team/{memberId}/wallets/{walletId}/provision
---

Provision a specific wallet for an organization team member, activating their access. Provide the member ID and wallet ID as path parameters. This operation completes the wallet assignment flow and enables the member to use the wallet within the organization context.
