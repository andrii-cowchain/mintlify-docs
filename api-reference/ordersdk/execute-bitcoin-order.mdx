title: "Execute a Fully Signed Bitcoin Multisig Order on-Chain"
sidebarTitle: "Execute Bitcoin Order"
description: "PATCH /sdk/order/execute/btc — Broadcasts a fully signed Bitcoin multisig order to the network and returns the on-chain transaction ID."
---

Use this endpoint to execute a Bitcoin order that has collected all required multisig signatures. The API finalizes the PSBT, broadcasts the transaction to the Bitcoin network, and returns the on-chain transaction ID along with the updated order status.
