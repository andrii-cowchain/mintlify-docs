title: "Get Cardano UTXOs Available for an ADA Multisig Order"
sidebarTitle: "Get ADA UTXOs"
description: "GET /sdk/order/utxos/ada — Retrieves the unspent transaction outputs (UTXOs) available for constructing a Cardano ADA multisig order transaction."
---

Use this endpoint to fetch the unspent transaction outputs (UTXOs) associated with a Cardano ADA order. The returned UTXO set provides the inputs needed to construct and balance the ADA transaction before it is signed by the multisig participants and submitted to the Cardano network.
