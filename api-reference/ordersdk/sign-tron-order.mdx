openapi: post /sdk/order/sign/tron
title: "Sign a Pending Tron Multisig Order via the Order SDK"
sidebarTitle: "Sign Tron Order"
description: "POST /sdk/order/sign/tron — Submits a signer's signature for a pending Tron multisig order, advancing it toward the required signing threshold."
---

Use this endpoint to record an individual signer's approval for a pending Tron order. Submit the order ID and the signer's cryptographic signature; once the required threshold of signatures is reached, the order becomes eligible for execution on the Tron network.
