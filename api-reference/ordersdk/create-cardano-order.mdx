title: "Create a New Cardano Multisig Order via the Order SDK"
sidebarTitle: "Create Cardano Order"
description: "POST /sdk/order/create/ada — Creates a new Cardano (ADA) multisig order, returning the order record with its ID and pending-signature status."
---

Use this endpoint to initiate a new order on the Cardano blockchain through the Order SDK. Provide the necessary ADA transaction parameters and the API will register the order, assign it a unique ID, and set its initial status to await the required multisig signatures before on-chain submission.
