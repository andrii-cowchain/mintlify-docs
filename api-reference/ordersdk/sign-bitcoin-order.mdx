openapi: post /sdk/order/sign/btc
title: "Sign a Pending Bitcoin Multisig Order via the Order SDK"
sidebarTitle: "Sign Bitcoin Order"
description: "POST /sdk/order/sign/btc — Submits a signer's signature for a pending Bitcoin multisig order, advancing it toward the required signing threshold."
---

Use this endpoint to record an individual signer's approval for a pending Bitcoin order. Submit the order ID and the signer's PSBT or raw signature; once the required threshold of signatures is collected, the order becomes eligible for broadcast to the Bitcoin network.
