openapi: post /sdk/order/reject/evm
title: "Reject a Pending EVM Multisig Order via the Order SDK"
sidebarTitle: "Reject EVM Order"
description: "POST /sdk/order/reject/evm — Records a signer's rejection of a pending EVM multisig order, preventing it from reaching the execution threshold."
---

Use this endpoint to register a signer's explicit rejection of a pending EVM order. Submit the order ID along with the rejecting signer's details; the order will be marked as rejected and will not proceed to execution on the EVM-compatible chain.
