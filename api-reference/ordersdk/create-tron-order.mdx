title: "Create a New Tron Multisig Order via the Order SDK"
sidebarTitle: "Create Tron Order"
description: "POST /sdk/order/create/tron — Creates a new Tron multisig order, returning the order record with its ID and pending-signature status."
---

Use this endpoint to initiate a new order on the Tron network through the Order SDK. Submit the required transaction parameters and the API will register the order, assign it a unique ID, and set its initial status to await the required number of signatures before execution.
