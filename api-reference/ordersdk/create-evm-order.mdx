title: "Create a New EVM Multisig Order via the Order SDK"
sidebarTitle: "Create EVM Order"
description: "POST /sdk/order/create/evm — Creates a new EVM-based multisig order, returning the order record with its ID and pending-signature status."
---

Use this endpoint to initiate a new order on an EVM-compatible chain through the Order SDK. Submit the required transaction parameters and the API will register the order, assign it a unique ID, and set its initial status to await the required number of signatures before execution.
