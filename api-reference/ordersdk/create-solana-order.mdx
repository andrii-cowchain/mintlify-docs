openapi: post /sdk/order/create/sol
title: "Create a New Solana Multisig Order via the Order SDK"
sidebarTitle: "Create Solana Order"
description: "POST /sdk/order/create/sol — Creates a new Solana multisig order, returning the order record with its ID and pending-signature status."
---

Use this endpoint to initiate a new order on the Solana blockchain through the Order SDK. Submit the required transaction parameters and the API will register the order, assign it a unique ID, and set its initial status to await the required number of signatures before execution.
