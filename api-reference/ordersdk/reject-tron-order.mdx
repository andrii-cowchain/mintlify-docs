---
openapi: post /sdk/order/reject/tron
title: "Reject a Pending Tron Multisig Order via the Order SDK"
sidebarTitle: "Reject Tron Order"
description: "POST /sdk/order/reject/tron — Records a signer's rejection of a pending Tron multisig order, preventing it from reaching the execution threshold."
---

Use this endpoint to register a signer's explicit rejection of a pending Tron order. Submit the order ID along with the rejecting signer's details; the order will be marked as rejected and will not proceed to execution on the Tron network.
