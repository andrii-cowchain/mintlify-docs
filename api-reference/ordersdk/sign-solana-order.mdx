openapi: post /sdk/order/sign/sol
title: "Sign a Pending Solana Multisig Order via the Order SDK"
sidebarTitle: "Sign Solana Order"
description: "POST /sdk/order/sign/sol — Submits a signer's signature for a pending Solana multisig order, advancing it toward the required signing threshold."
---

Use this endpoint to record an individual signer's approval for a pending Solana order. Submit the order ID and the signer's cryptographic signature; once the required threshold of signatures is reached, the order becomes eligible for execution on the Solana network.
