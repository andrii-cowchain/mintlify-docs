title: "Sign a Pending XRP Ledger Multisig Order via the SDK"
sidebarTitle: "Sign XRP Order"
description: "POST /sdk/order/sign/xrp — Submits a signer's signature for a pending XRP Ledger multisig order, advancing it toward the required signing threshold."
---

Use this endpoint to record an individual signer's approval for a pending XRP Ledger order. Submit the order ID and the signer's cryptographic signature; once the required threshold of signatures is reached, the order becomes eligible for submission to the XRP Ledger.
