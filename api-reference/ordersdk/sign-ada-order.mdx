openapi: post /sdk/order/sign/ada
title: "Sign a Pending Cardano ADA Order via the Order SDK"
sidebarTitle: "Sign ADA Order"
description: "POST /sdk/order/sign/ada — Submits a signer's signature for a pending Cardano ADA multisig order, advancing it toward the required signing threshold."
---

Use this endpoint to record an individual signer's approval for a pending Cardano ADA order. Submit the order ID and the signer's cryptographic signature; once the required threshold of signatures is collected, the order becomes eligible for broadcast to the Cardano network.
