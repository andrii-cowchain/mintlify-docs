title: "Execute a Fully Signed Solana Multisig Order on-Chain"
sidebarTitle: "Execute Solana Order"
description: "PATCH /sdk/order/execute/sol — Broadcasts a fully signed Solana multisig order to the network and returns the on-chain transaction signature."
---

Use this endpoint to execute a Solana order that has gathered all required signatures. The API assembles the signed transaction and broadcasts it to the Solana network, returning the on-chain transaction signature and updating the order status to executed.
