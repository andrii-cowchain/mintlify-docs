---
openapi: post /sdk/order/activate/sol
title: "Activate a Pending Solana Order via the Order SDK"
sidebarTitle: "Activate Solana Order"
description: "POST /sdk/order/activate/sol — Activates a previously created Solana multisig order, transitioning it to active status and enabling signing."
---

Use this endpoint to activate a Solana order that has already been created through the Order SDK. Once activated, the order becomes available for signers to review and submit their signatures before final execution on-chain.
