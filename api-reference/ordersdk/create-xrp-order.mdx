title: "Create a New XRP Ledger Multisig Order via the Order SDK"
sidebarTitle: "Create XRP Order"
description: "POST /sdk/order/create/xrp — Creates a new XRP Ledger multisig order, returning the order record with its ID and pending-signature status."
---

Use this endpoint to initiate a new order on the XRP Ledger through the Order SDK. Submit the required transaction parameters and the API will register the order, assign it a unique ID, and set its initial status to await the required number of signatures before submission to the ledger.
