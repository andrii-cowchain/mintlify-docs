openapi: post /sdk/order/send/ada
title: "Send and Broadcast a Fully Signed Cardano ADA Order"
sidebarTitle: "Send ADA Order"
description: "POST /sdk/order/send/ada — Submits a fully signed Cardano ADA multisig transaction to the network and returns the resulting transaction hash."
---

Use this endpoint to broadcast a fully signed Cardano ADA order to the network. Once all required multisig signatures have been collected and the transaction is assembled, call this endpoint to submit it to the Cardano blockchain and receive the on-chain transaction hash in response.
