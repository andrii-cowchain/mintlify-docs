title: "Sign a Pending EVM Multisig Order via the Order SDK"
sidebarTitle: "Sign EVM Order"
description: "POST /sdk/order/sign/evm — Submits a signer's signature for a pending EVM multisig order, advancing it toward the required signing threshold."
---

Use this endpoint to record an individual signer's approval for a pending EVM order. Submit the order ID and the signer's cryptographic signature; once the required threshold of signatures is reached, the order becomes eligible for execution on the EVM-compatible chain.
