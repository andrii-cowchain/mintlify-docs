title: "Execute a Fully Signed XRP Ledger Multisig Order"
sidebarTitle: "Execute XRP Order"
description: "PATCH /sdk/order/execute/xrp — Executes a fully signed XRP Ledger multisig order by submitting the transaction and returning the ledger transaction hash."
---

Use this endpoint to execute an XRP Ledger order that has gathered all required signatures. The API combines the collected signatures, submits the multi-signed transaction to the XRP Ledger, and returns the resulting transaction hash along with the updated order status.
