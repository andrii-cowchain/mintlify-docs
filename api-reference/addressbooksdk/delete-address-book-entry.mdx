---
title: "Permanently Delete an Address Book Entry by Record ID"
sidebarTitle: "Delete Entry"
description: "DELETE /sdk/address-book/{id} — Permanently delete an address book entry by its ID, removing the contact record from the user's address book."
openapi: delete /sdk/address-book/{id}
---

Permanently delete an address book entry from the authenticated user's address book. Provide the entry ID as a path parameter. Once deleted, the contact record is removed and cannot be recovered.
