---
title: "Resolve the Type of a Blockchain Address by Chain Type"
sidebarTitle: "Get Address Type"
description: "GET /sdk/address-book/type/{chainType}/{address} — Classify a blockchain address by chain type, returning whether it is a wallet, multisig, or contract."
openapi: get /sdk/address-book/type/{chainType}/{address}
---

Resolve the classification type of a blockchain address for a given chain. Provide the chain type and the target address as path parameters; the endpoint returns the address type, indicating whether the address belongs to a standard wallet, multisig, contract, or another category.
