---
title: "Retrieve All Address Book Entries for the Current User"
sidebarTitle: "Get Entries"
description: "GET /sdk/address-book — Retrieve all address book entries for the authenticated user, returning saved contacts with names, addresses, and chain types."
openapi: get /sdk/address-book
---

Retrieve all address book entries belonging to the authenticated user. The response returns a list of saved contacts, each including the entry ID, display name, blockchain address, and associated chain type.
