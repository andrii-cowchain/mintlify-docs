---
openapi: get /sdk/wallets/pubkey
title: "Retrieve the Recovery Public Key for the Current User"
sidebarTitle: "Recovery Public Key"
description: "GET /sdk/wallets/pubkey - Returns the recovery public key linked to the authenticated user's account for use in backup and key-share recovery flows."
---

Retrieve the recovery public key linked to the authenticated user's account. This public key is used in wallet backup and key-share recovery flows to verify the identity of the key holder. The response contains the encoded public key and its associated algorithm metadata.
