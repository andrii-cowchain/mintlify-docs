title: "Get the Aggregated Total Asset Value for All Wallets"
sidebarTitle: "Total Asset Value"
description: "GET /sdk/wallets/total-asset-value - Returns the combined USD value of all assets held across every wallet owned by the authenticated user."
---

Calculate the combined USD value of all assets held across every wallet belonging to the authenticated user. This endpoint aggregates balances from all chains and tokens into a single total figure, making it convenient for displaying an overall portfolio value on a summary screen.
