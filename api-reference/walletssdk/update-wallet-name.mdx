---
openapi: post /sdk/wallets/update/name
title: "Update the Human-Readable Display Name of a Wallet"
sidebarTitle: "Update Name"
description: "POST /sdk/wallets/update/name - Updates the display name of an existing wallet by accepting the wallet address and new name in the request body."
---

Update the display name assigned to an existing wallet. Supply the wallet address and the desired new name in the JSON request body. Only the authenticated owner of the wallet may rename it. The updated wallet object is returned on success.
